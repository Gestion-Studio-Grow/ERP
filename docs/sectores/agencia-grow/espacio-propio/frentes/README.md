# frentes/ — trabajo por disciplina del sector

1 frente = 1 sesión aislada (metodología del repo). Subcarpetas sugeridas:
- `creativo/` — identidad, dirección de arte, piezas, branding por marca.
- `performance-ads/` — campañas paid (Meta/Google/TikTok), tracking, medición.
- `contenido/` — social, copy, calendario editorial, video/foto.
- `front-consola-grow/` — diseño de producto: la Consola Grow + storefronts brandeables.

Cada frente entrega en su rama; el PMO integra. Nada de esto toca el Core del ERP.
