# clientes/ — entregables por cliente (delivery)

Una carpeta por cliente (delivery, regla 4: no toca el core compartido). Guarda briefs, assets y
entregables de campañas/servicios. El cliente que necesita ERP/tienda se resuelve dando de **alta un
tenant** del ERP (puente ADR-028), no con código a medida acá.
